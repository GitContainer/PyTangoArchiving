Snap Application

Description

Snap is an application created for managing contexts and snapshots.
It eables the user to create/modify/delete contexts, save/load/delete snapshots and export values of snapshots' attributes to CSV files.
Additionally applciation is ewuipped in compare function which allows the user to compare values between two snapsoth or between snapshot and actual values.

Application can be downloaded from here

Program

1. Select context drop-down list

2. New context button (Create Context dialog appears)

3. Edit selected context (Modify Context dialog appears)

4. Save snapshot button (Save dialog appears)

5. Load snapshot button (Load dialog appears)

6. View mode [table view(static), live view, compare view]

7. Export to CSV button (CSV dialog appears)

8. Close application
